Photolithography & Optical Metrology Simulation

Advanced Semiconductor Manufacturing Process Modeling

Project Overview

This project models and simulates key steps in semiconductor photolithography and optical metrology workflows, the backbone of IC manufacturing. The comprehensive simulation framework includes mask pattern generation, optical proximity correction (OPC), and defect inspection analysis, providing insights into the complex physics governing nanometer-scale patterning processes.

Key Objectives

  • Model complete photolithography process chain
  • Simulate optical proximity effects at sub-wavelength scales
  • Analyze resolution limits under DUV/EUV sources
  • Optimize OPC algorithms for critical dimension control
  • Develop metrology signal models for process monitoring
  • Create defect detection algorithms for wafer inspection

Methods & Approach

  • Mask Simulation: Monte Carlo & Double Gaussian modeling
  • Optical Analysis: Fourier optics for resolution limits
  • OPC Implementation: Rule-based and model-based corrections
  • Metrology: Scatterometry & interferometry modeling
  • Defect Detection: Edge detection with noise filtering
  • Process Window: Focus-exposure matrix optimization

Technologies & Tools

Technical Implementation

Optical System Modeling

Developed comprehensive models for DUV (193nm) and EUV (13.5nm) lithography systems, including partial coherence effects, aberrations, and pupil filtering. Implemented Hopkins' method for aerial image calculation with optimized computational efficiency.

2

Mask Pattern Generation

Created algorithms for mask pattern synthesis including sub-resolution assist features (SRAF), phase shift masks (PSM), and source mask optimization (SMO). Implemented both rule-based and model-based OPC with iterative convergence.

3

Resist Process Simulation

Modeled photoresist exposure and development using continuum and stochastic approaches. Incorporated effects of photoacid diffusion, quencher distribution, and line-edge roughness for accurate CD prediction.

4

Metrology & Inspection

Developed scatterometry models for CD and overlay measurement, interferometry simulations for film thickness, and machine learning-based defect classification algorithms for automated wafer inspection.

Aerial Image Calculation (Python)

# Fourier optics-based aerial image simulation
def calculate_aerial_image(mask, NA, wavelength, sigma):
    """
    Calculate aerial image using Hopkins' method
    Includes partial coherence and pupil function
    """
    # Fourier transform of mask pattern
    mask_spectrum = np.fft.fft2(mask)
    
    # Generate source points for partial coherence
    source_points = generate_source_distribution(sigma)
    
    # Sum coherent systems for each source point
    aerial_image = np.zeros_like(mask, dtype=float)
    for point in source_points:
        pupil = generate_pupil_function(NA, wavelength, point)
        coherent_image = inverse_fourier(mask_spectrum * pupil)
        aerial_image += np.abs(coherent_image)**2
        
    return aerial_image / len(source_points)

OPC Algorithm Implementation

# Model-based OPC with iterative convergence
def apply_opc_correction(target_pattern, model, iterations=10):
    """
    Iterative OPC to achieve target pattern on wafer
    Uses process model for forward simulation
    """
    mask = target_pattern.copy()
    
    for i in range(iterations):
        # Forward simulation through process model
        wafer_image = model.simulate(mask)
        
        # Calculate error between target and simulated
        error = target_pattern - wafer_image
        
        # Update mask using gradient descent
        gradient = model.calculate_gradient(mask, error)
        mask += learning_rate * gradient
        
        # Apply constraints (min feature size, MRC)
        mask = apply_mask_constraints(mask)
        
        if np.mean(np.abs(error)) < tolerance:
            break
            
    return mask
