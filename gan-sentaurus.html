TCAD Simulation Files

HEMT Thermal Command File

Main simulation command file for thermal analysis of InGaN/GaN HEMTs.

# HEMT Thermal Simulation Command File
# Device: InGaN/GaN HEMT with Field Plate
# Substrate: SiC

Device HEMT {
    File {
        Grid = "hemt_structure.grd"
        Plot = "hemt_thermal.tdr"
        Current = "hemt_thermal.plt"
        Parameter = "hemt_material.par"
    }
    
    Electrode {
        { Name="source" Voltage=0.0 }
        { Name="drain" Voltage=20.0 }
        { Name="gate" Voltage=-2.0 }
        { Name="substrate" Voltage=0.0 }
    }
    
    Physics {
        Temperature = 300
        Thermode
        Thermodynamic
        EffectiveIntrinsicDensity(BandGapNarrowing(OldSlotboom))
        Mobility(
            DopingDep
            eHighFieldSaturation(GradQuasiFermi)
            hHighFieldSaturation(GradQuasiFermi)
        )
        Recombination(
            SRH(DopingDep TempDependence)
            Auger
            Radiative
        )
    }
}

# Temperature-dependent properties included

        epsilon = 9.5

        # Temperature dependent: k(T) = k0 * (300/T)^1.4

Simulation Results

Key Findings from TCAD Analysis

Temperature Distribution

Peak temperature occurs at the drain-side edge of the gate with a hotspot of 127°C at 2W power dissipation.

Electric Field Profile

Maximum electric field of 3.5 MV/cm at gate edge, reduced to 2.8 MV/cm with field plate design.

Current Density

2DEG sheet carrier density of 1.2×10¹³ cm⁻² with mobility of 1500 cm²/V·s at room temperature.

Thermal Resistance

Junction-to-case thermal resistance of 50 K/W achieved with optimized SiC substrate integration.

Running TCAD Simulations

Execution Commands

# Structure generation
sde -l sde.log hemt_structure.cmd

# Device simulation
sdevice -l sdevice.log hemt_thermal.cmd

# Visualization
svisual hemt_thermal.tdr

# Parameter extraction
inspect hemt_thermal.plt
