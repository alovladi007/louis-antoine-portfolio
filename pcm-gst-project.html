Phase Change Memory (PCM) Research

Advanced Non-Volatile Memory Technology

Project Overview

Comprehensive simulation framework for Phase Change Memory devices based on Ge-Sb-Te (GST) chalcogenide materials. This research project implements advanced materials science models and device physics to simulate and optimize next-generation non-volatile memory technology.

Technologies & Methods

Python NumPy/SciPy JMAK Model Electro-thermal Modeling Monte Carlo Materials Science TCAD Jupyter

Key Features

JMAK Crystallization Kinetics

Johnson-Mehl-Avrami-Kolmogorov model for temperature-dependent phase transformation with activation energy of 2.1 eV.

Electro-thermal Device Model

Coupled electrical and thermal dynamics with Joule heating, percolation-based resistance mixing, and lumped thermal model.

Pulse Operations

RESET (100ns @ 3.5V) for amorphization and SET (500ns @ 1.8V) for crystallization with real-time phase tracking.

Threshold Switching

Quasi-static I-V characteristics with hysteresis, state-dependent threshold voltage, and hold current mechanism.

Reliability Analysis

Arrhenius retention model and Weibull endurance distribution for 10-year data retention and 10⁶ cycle lifetime.

Monte Carlo Variability

Statistical analysis with ±10% parameter variations for process tolerance and yield optimization.

Technical Implementation

Material Properties

  • Active layer: 50 nm Ge-Sb-Te (GST)
  • Resistance ratio: >1000× (amorphous/crystalline)
  • Melting temperature: 900 K
  • Crystallization onset: 430 K
  • Thermal capacitance: 10⁻¹¹ J/K
  • Thermal resistance: 5×10⁵ K/W

Simulation Capabilities

  • Temperature-dependent crystallization kinetics
  • Resistance vs crystalline fraction relationships
  • Pulse-driven phase transitions with sticky amorphization
  • 2D parameter optimization landscapes
  • Comprehensive data visualization and analysis

Code Sample

# PCM Device Simulation Example
from pcm.device import PCMDevice
from pcm.utils import plot_pulse_results

# Initialize device with GST parameters
device = PCMDevice()

# Simulate RESET pulse (amorphization)
t_reset, V_reset = device.create_reset_pulse(duration_s=100e-9, voltage_V=3.5)
result_reset = device.simulate_voltage_pulse(V_reset, t_reset, X0=1.0)

# Simulate SET pulse (crystallization)
t_set, V_set = device.create_set_pulse(duration_s=500e-9, voltage_V=1.8)
result_set = device.simulate_voltage_pulse(V_set, t_set, X0=0.0)

print(f"RESET: X = {result_reset['X'][0]:.3f} → {result_reset['X'][-1]:.3f}")
print(f"SET: X = {result_set['X'][0]:.3f} → {result_set['X'][-1]:.3f}")
print(f"Resistance ratio: {result_reset['R_ohm'][-1]/result_set['R_ohm'][-1]:.0f}×")
