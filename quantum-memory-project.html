Quantum Memory System for EIT

Electromagnetically-Induced Transparency in ⁸⁷Rb

Project Overview

Comprehensive computational analysis and experimental simulation of Electromagnetically-Induced Transparency (EIT) in ⁸⁷Rb atomic vapor for quantum memory applications. This project implements advanced quantum optics simulations to demonstrate coherent light storage and retrieval with high efficiency.

Key Features

  • Λ-type three-level atomic system modeling in ⁸⁷Rb
  • Real-time EIT spectroscopy simulation
  • Quantum memory storage and retrieval dynamics
  • Group velocity control and slow light effects
  • Comprehensive parameter optimization
  • Comparison with experimental literature

Key Results

Storage Performance

  • Storage efficiency: 93.6% at optimal parameters
  • Coherence time: 100 μs (limited by ground state decoherence)
  • Time-bandwidth product: Bτ ≈ 100

EIT Parameters

  • Peak transmission: 99.8% on two-photon resonance
  • EIT linewidth: 1.5 MHz at Ωc = 3 MHz
  • Group delay: 10.6 μs for OD = 100
