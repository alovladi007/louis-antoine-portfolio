1. Fundamental Principles

1.1 Rayleigh Resolution Criterion

The fundamental limit of optical lithography is described by the Rayleigh criterion:

Rayleigh Resolution

\[ R = k_1 \frac{\lambda}{NA} \]

where:

  • R = minimum resolvable feature size
  • k₁ = process factor (typically 0.25-0.8)
  • λ = wavelength of light
  • NA = numerical aperture

1.2 Depth of Focus

The depth of focus determines the process latitude in the vertical direction:

Depth of Focus

\[ DOF = k_2 \frac{\lambda}{NA^2} \]

where k₂ is typically around 0.5-1.0

Key Insight: There's a fundamental trade-off between resolution and DOF. Increasing NA improves resolution but reduces DOF quadratically.

1.3 Numerical Aperture

The numerical aperture defines the light-gathering ability of the optical system:

\[ NA = n \sin(\theta_{max}) \]

where n is the refractive index of the medium and θ_max is the maximum half-angle of the cone of light

Technology Wavelength Typical NA Resolution
i-line 365 nm 0.5-0.6 ~350 nm
KrF 248 nm 0.6-0.8 ~180 nm
ArF 193 nm 0.75-0.93 ~90 nm
ArF immersion 193 nm 1.2-1.35 ~45 nm
EUV 13.5 nm 0.33 ~13 nm

2. Optical Theory

2.1 Hopkins Imaging Theory

Partially coherent imaging is described by Hopkins' theory using Transmission Cross Coefficients (TCCs):

Aerial Image Intensity

\[ I(x,y) = \sum_{m,n} \sum_{p,q} TCC_{m,n,p,q} \cdot M_m M_n^* M_p^* M_q \]

where M represents the mask spectrum coefficients

2.2 Partial Coherence

The degree of spatial coherence is characterized by the coherence factor σ:

\[ \sigma = \frac{NA_{illumination}}{NA_{imaging}} \]

Coherent (σ = 0)

• Highest contrast
• Severe ringing
• Poor for dense features

Partial (σ = 0.3-0.8)

• Balanced performance
• Good process window
• Industry standard

Incoherent (σ > 1)

• Low contrast
• No ringing
• Good for isolated features

2.3 Fourier Optics

The imaging process can be understood through Fourier analysis:

Pupil Function

\[ P(f_x, f_y) = \begin{cases} 1 & \text{if } \sqrt{f_x^2 + f_y^2} \leq \frac{NA}{\lambda} \\ 0 & \text{otherwise} \end{cases} \]

2.4 Aberrations

Lens aberrations are described using Zernike polynomials:

Wavefront Error

\[ W(\rho, \theta) = \sum_{n,m} a_{nm} Z_n^m(\rho, \theta) \]

Zernike Term Name Effect
Z₄ Defocus Shifts best focus position
Z₅, Z₆ Astigmatism Different focus in X/Y
Z₇, Z₈ Coma Pattern placement error
Z₉ Spherical Reduces contrast

3. Resolution Enhancement Technologies

3.1 Off-Axis Illumination (OAI)

Modified illumination improves resolution and DOF for specific pattern types:

Annular Illumination

• σ_inner = 0.5-0.7
• σ_outer = 0.8-0.95
• Good for various pitches

Dipole Illumination

• Two poles at ±45° or 0°/90°
• Excellent for lines/spaces
• k₁ can reach 0.25

Quadrupole

• Four poles at corners
• Good for Manhattan geometry
• Balanced X/Y performance

3.2 Phase Shift Masks (PSM)

Phase Shift Principle

\[ E_{total} = E_1 + E_2 e^{i\pi} = E_1 - E_2 = 0 \]

Destructive interference creates dark regions with higher contrast

3.3 Sub-Resolution Assist Features (SRAF)

Non-printing features that improve the process window of main features:

Design Rules for SRAF:
  • Width: 0.3-0.4 × minimum feature size
  • Spacing: 0.8-1.2 × minimum pitch
  • Must not print under any process condition

4. Optical Proximity Correction

4.1 Edge Placement Error

EPE Definition

\[ EPE = |CD_{printed} - CD_{target}| \]

4.2 Model-Based OPC

Iterative correction based on process models:

OPC Algorithm Steps:
  1. Simulate aerial image from current mask
  2. Apply resist model to predict printed pattern
  3. Calculate EPE at all edges
  4. Move edges to compensate: Δ = -K × EPE
  5. Repeat until EPE < tolerance

4.3 Correction Types

Correction Type Application Typical Bias
Line-end shortening Extend line ends 10-30 nm
Corner rounding Add serifs 5-15 nm
Iso-dense bias Different CD for isolated/dense 5-20 nm
Gate CD control Critical dimension uniformity ±2 nm

5. Process Window

5.1 Bossung Curves

CD variation through focus at different exposure doses:

Bossung Curve Equation

\[ CD(z) = CD_0 + a \cdot z^2 + b \cdot z^4 \]

where z is the defocus distance

5.2 Process Window Metrics

Exposure Latitude (EL)

\[ EL = \frac{E_{max} - E_{min}}{E_{nominal}} \times 100\% \]

Depth of Focus (DOF)

\[ DOF = Z_{max} - Z_{min} \]

Normalized Image Log-Slope

\[ NILS = \frac{dI/dx}{I} \cdot CD \]

5.3 Process Capability

Process Capability Index

\[ C_{pk} = \min\left(\frac{USL - \mu}{3\sigma}, \frac{\mu - LSL}{3\sigma}\right) \]

Target Values:
  • C_pk > 1.33: Capable process
  • C_pk > 1.67: Well-controlled process
  • C_pk > 2.0: Six Sigma quality

6.1 Defect Types

Defect Type Cause Detection Method

6.2 Yield Model

Poisson Yield Model

\[ Y = e^{-AD_0} \]

where A is the chip area and D₀ is the defect density

7. Metrology Principles

7.1 Critical Dimension Measurement

CD-SEM

• Resolution: 1-2 nm
• Non-destructive
• Charging effects possible

Scatterometry

• Fast, non-contact
• Profile information
• Requires modeling

AFM

• 3D profile
• Sub-nm resolution
• Slow, small area

7.2 Overlay Measurement

Total Overlay Error

\[ OVL_{total} = \sqrt{OVL_x^2 + OVL_y^2 + 3\sigma_{random}^2} \]

8. Advanced Topics

8.1 EUV Lithography

EUV Challenges:
  • Reflective optics (no transmissive materials)
  • Vacuum operation required
  • Stochastic effects at low photon counts
  • Mask 3D effects
  • Source power limitations

8.2 Multiple Patterning

Double Patterning Techniques:
  • LELE: Litho-Etch-Litho-Etch
  • SADP: Self-Aligned Double Patterning
  • SAQP: Self-Aligned Quadruple Patterning

8.3 Computational Lithography

Inverse Lithography Technology (ILT)

Optimization problem to find ideal mask:

\[ M_{opt} = \arg\min_M ||I(M) - I_{target}||^2 + \lambda R(M) \]

where R(M) is a regularization term for manufacturability
