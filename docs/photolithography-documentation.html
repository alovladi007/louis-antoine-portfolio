1. Overview

Introduction to the Photolithography Simulation Suite

The Photolithography & Optical Metrology Simulation Suite is a comprehensive web-based platform for simulating key steps in semiconductor photolithography and optical metrology workflows. This documentation covers all features, tools, and algorithms available in the suite.

Key Capabilities

  • 9 Interactive Demonstrations: Real-time simulations with adjustable parameters
  • 6 Core Algorithms: Advanced computational methods with visualizations
  • Live Process Simulator: Real-time aerial image calculation
  • Python Backend: Scientific computing with NumPy, SciPy, and OpenCV
  • Web Interface: Interactive visualizations using Plotly.js and Three.js

3. Core Algorithms

Mathematical foundations and implementations

Hopkins Imaging Theory

File: hopkins-imaging-sim.html

Mathematical Foundation:

I(x,y) = ∫∫ TCC(f₁,f₂) · M(f₁) · M*(f₂) · exp[2πi(f₁-f₂)·(x,y)] df₁df₂ Where: - TCC = Transmission Cross Coefficient - M = Mask spectrum - f₁, f₂ = Spatial frequencies

Implementation Features:

  • TCC Matrix: Pre-computed for speed
  • Source Shapes: Conventional, annular, dipole, quadrupole
  • Coherence: Partial coherence factor σ = 0.1 to 1.0

Abbe-Rayleigh Diffraction Theory

File: abbe-rayleigh-sim.html

Key Equations:

Resolution = 0.61 × λ / NA (Rayleigh Criterion) Diffraction Order: m × λ = d × sin(θ) (Grating Equation)

Vector Calculations:

  • TE Polarization: Electric field perpendicular to incidence plane
  • TM Polarization: Electric field parallel to incidence plane
  • Unpolarized: Average of TE and TM

Kirchhoff Approximation

File: kirchhoff-sim.html

Mask Models:

  • Thin Mask: 2D amplitude/phase transmission
  • Thick Mask: 3D electromagnetic scattering
  • EUV Masks: Multilayer reflective stack with absorber

SOCS Decomposition

File: socs-decomposition-sim.html

Algorithm:

1. Decompose TCC matrix into eigenvalues/eigenvectors 2. Select dominant kernels (typically 4-10) 3. Calculate coherent systems for each kernel 4. Sum intensities for final aerial image

Performance:

  • CPU: 10x faster than Hopkins
  • GPU: 50x faster with CUDA acceleration

Neural OPC

File: neural-opc-sim.html

Network Architecture:

Input Layer (256×256 mask pattern) ↓ Conv2D (64 filters, 3×3) ↓ MaxPool2D (2×2) ↓ Conv2D (128 filters, 3×3) ↓ MaxPool2D (2×2) ↓ Dense (512 units) ↓ Output Layer (256×256 corrected mask)

Training Data:

  • 1M+ pattern-correction pairs
  • Data augmentation: rotation, flipping, scaling
  • Loss function: MSE + edge placement error

Zernike Polynomials

File: zernike-polynomials-sim.html

Common Aberrations:

Zernike Name Effect
Z4 Defocus Uniform blur
Z5, Z6 Astigmatism Directional blur
Z7, Z8 Coma Comet-like distortion
Z9 Spherical Edge vs center focus

Getting Started

Quick Start Guide

Step 1: Access the Main Suite

Navigate to photolithography-comprehensive.html to access the main interface.

Step 2: Explore Interactive Demos

Click on any demo card in the "Interactive Demonstrations" section to launch simulations.

Step 3: Try Core Algorithms

Visit the Algorithms Hub to explore mathematical foundations with live visualizations.

Step 4: Run Python Backend

# Clone repository git clone https://github.com/yourusername/photolithography-simulation.git # Install dependencies pip install -r requirements.txt # Run Streamlit dashboard streamlit run streamlit_app.py # Or use Jupyter notebooks jupyter notebook notebooks/

Step 5: Customize Parameters

All simulations have adjustable parameters. Experiment with different settings to understand their effects.

Recommended Learning Path:

  1. Start with Mask Pattern Generation to understand basics
  2. Move to Fourier Optics for optical theory
  3. Explore OPC for pattern correction
  4. Try Monte Carlo for statistical analysis
