Photolithography & Optical Metrology Simulation

Mask modeling, OPC, Fourier optics, and interferometry for nanometer-scale patterning.

Python MATLAB Fourier Optics OPC Scatterometry

Overview

This project simulates lithography imaging (Monte Carlo vs. double Gaussian PSF), explores OPC trade-offs, and models metrology signals (scatterometry/interferometry) to estimate thin-film and linewidth parameters.

Methods & Tools

  • Mask pattern generator with noise/LER controls
  • Fourier optics PSF/MTF modeling across DUV/EUV (248 nm, 193 nm, 13.5 nm)
  • OPC evaluation (serifs, hammerheads) and aerial image analysis
  • Interferometry phase retrieval and thickness estimation
  • Automated defect/edge detection for simulated wafer maps

Results & Visuals

  • Resolution improvement curve vs. wavelength/NA
  • OPC reduces corner rounding and improves CD control
  • Thin-film thickness error < 1.5% in synthetic benchmarks

Resolution vs. Exposure Wavelength (illustrative)

Impact

Demonstrates end-to-end understanding of the lithography + metrology loop—skills directly relevant to ASML/KLA tool chains and fab process integration.

Defect Detection & Yield Prediction with Machine Learning

CNNs for defect classification + Bayesian yield forecasting with interactive dashboards.

PyTorch CNN Bayesian ML Streamlit MLOps

Overview

A full ML pipeline for wafer/SEM image defect classification and lot-level yield prediction. Includes uncertainty-aware inference for rare defects and an ops-friendly dashboard for fab engineers.

Methods & Tools

  • Dataset curation (public SEM/wafer images + synthetic augmentation)
  • Transfer learning (ResNet/EfficientNet) and class-imbalance handling
  • Bayesian neural networks for uncertainty & active learning triggers
  • Feature store + experiment tracking (MLflow)
  • Streamlit dashboard with confusion matrices and lot drilldowns

Results & Visuals

  • Top-1 accuracy > 95% on balanced test set; AUROC > 0.98
  • Uncertainty thresholds reduce false positives by ~30% in rare classes
  • Yield forecast sMAPE < 4% on holdout lots

Yield by Lot with Confidence Intervals (illustrative)

Impact

Shows measurable wins in inspection throughput and yield predictability—high-value skills for advanced fabs and equipment vendors.
