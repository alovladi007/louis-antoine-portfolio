Waveform Characteristics

The UART transmitter operates at 115200 baud with a standard 8N1 configuration. Each character transmission consists of a start bit (logic 0), 8 data bits transmitted LSB-first, and a stop bit (logic 1), totaling 10 bits per character.

Performance Model

The CPI (Cycles Per Instruction) model incorporates both data hazards and control hazards:

CPI = 1 + f_load-use × 1 + f_mispredict × P

Where:
  - f_load-use: Fraction of load-use stalls
  - f_mispredict: Branch mispredict rate
  - P: Branch penalty (2 cycles)

The analysis shows that even modest mispredict rates (10-20%) can significantly impact performance, highlighting the importance of branch prediction in achieving target throughput.

Performance Scaling

With a baseline CPI of 1.4, the processor achieves linear throughput scaling with frequency. Target FPGA implementations at 50-100 MHz deliver 35-70 MIPS, suitable for embedded control applications.
