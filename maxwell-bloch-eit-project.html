Overview

This project implements a complete simulation framework for quantum optics experiments based on the Maxwell-Bloch equations, incorporating realistic physical effects such as Doppler broadening, ground-state decoherence, pulse storage/retrieval sequences, and control-field geometry (co- vs counter-propagating).

Key Features

Numerical Integrator Comparison

Euler vs 4th-order Runge-Kutta (RK4) solvers evaluated on large spatial/temporal grids to quantify stability and accuracy in stiff regimes.

Retrieval Efficiency Mapping

Heatmap analysis of EIT retrieval efficiency as a function of control-field ramp duration and ground-state dephasing rate γsg.

Geometry & Doppler Analysis

Comparative study of co- and counter-propagating control configurations across multiple Doppler widths, showing their impact on probe transmission dynamics.

Technical Implementation

The simulation solves the coupled Maxwell-Bloch equations for a three-level Λ-system, where:

  • Probe field: Couples |g⟩ → |e⟩ transition
  • Control field: Couples |s⟩ → |e⟩ transition
  • EIT window: Created by destructive quantum interference

  • Parameter optimization for cold- and warm-atom EIT systems

Usage Example

# Run all three main analyses python src/high_fidelity_qo_suite.py \ --outdir output \ --jpeg \ --euler_rk4 \ --heatmap \ --cocounter # High-fidelity simulation python src/high_fidelity_qo_suite.py \ --er_Nt 5000 --er_Nz 200 --er_Nv 15 \ --hm_Nt 4000 --hm_Nz 180 --hm_doppler \ --cc_Nt 4000 --cc_Nz 180 --cc_Nv 15

Simulation Parameters

Euler vs RK4 Analysis

  • --er_Nt: Time points (default: 3000)
  • --er_Nz: Spatial points (default: 120)
  • --er_Nv: Velocity classes (default: 9)
  • --er_sigma_v: Doppler width (default: 180 m/s)

Retrieval Heatmap

  • --hm_ramps_us: Control ramp durations
  • --hm_gammas_kHz: Dephasing rates
  • --hm_Nt: Time resolution (default: 2500)
  • --hm_doppler: Include Doppler effects

Co/Counter Comparison

  • --cc_sigmas: Doppler widths to test
  • --cc_Nt: Time points (default: 2500)
  • --cc_Nz: Spatial points (default: 100)
  • --cc_Nv: Velocity classes (default: 9)
