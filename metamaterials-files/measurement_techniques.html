Characterization Methods

THz Time-Domain Spectroscopy (THz-TDS)

• Broadband characterization from 0.1-10 THz

• Direct measurement of amplitude and phase

• Parameter extraction using NRW method

Near-Field Scanning Optical Microscopy (SNOM)

• Sub-wavelength resolution imaging

• Local field enhancement mapping

• Plasmonic mode visualization

Vector Network Analyzer (VNA)

• S-parameter measurement up to 110 GHz

• Free-space measurement setup

• Time-domain gating for isolation
