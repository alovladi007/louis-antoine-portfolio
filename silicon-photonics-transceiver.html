Project Overview

Designed and simulated a complete silicon photonics transceiver system operating at 25-50 Gbps for high-speed data center interconnects. This project demonstrates the integration of photonic components including modulators, photodetectors, and passive routing elements on a silicon-on-insulator (SOI) platform, showcasing expertise in semiconductor-optics-RF packaging crossover technologies.

Technical Implementation

System Architecture

  • Silicon Mach-Zehnder modulator with traveling wave electrodes
  • Germanium-on-silicon photodetector with responsivity > 1 A/W
  • Edge coupling with inverse taper mode converters
  • Integrated driver and TIA circuits for electrical interfacing
  • Temperature compensation using integrated heaters
